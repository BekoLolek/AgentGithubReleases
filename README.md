# ConfigTool Agent Plugin

The official Minecraft server plugin for ConfigTool - enabling real-time configuration synchronization.

## Download

Download the latest version from the [Releases](https://github.com/BekoLolek/AgentGithubReleases/releases) page.

## Supported Versions

### Minecraft Versions
- 1.20.4
- 1.20.6
- 1.21
- 1.21.1
- 1.21.4

### Server Software
- Spigot
- Paper
- Purpur
- Folia

## Installation

1. Download the latest JAR from releases
2. Place in your server's `plugins` folder
3. Start the server once to generate config
4. Edit `plugins/ConfigTool/config.yml` and add your server token
5. Restart the server

## Configuration

```yaml
# plugins/ConfigTool/config.yml
token: "your-server-token-here"
api-url: "https://api.configtool.org"
```

Get your server token from the [ConfigTool Dashboard](https://configtool.org/dashboard).
